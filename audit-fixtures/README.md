# Audit Fixtures

These files are synthetic fixtures for repeatable manual production audits.

- `journey-2-payment-reminder.html`
- `journey-2-payment-reminder.pdf`
- `journey-3-service-notice.txt`
- `journey-3-service-notice.docx`

They contain no personal data.

The Journey 2 files describe a fictional payment reminder from Greenfield Water
Services. They are used to check local PDF intake, payment-reminder assessment,
evidence export, and money-safety boundaries.

The Journey 3 files describe a fictional broadband price-change notice from
Northbridge Broadband. They are used to check local TXT and DOCX intake,
provider extraction, readable-file handling, and useful recovery guidance when
a document cannot be read.
